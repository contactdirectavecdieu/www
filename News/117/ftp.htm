If you do not have FTP software, please click twice on the following web site to download a FTP software.

ftp://ftp.ipswitch.com/pub/win32/ws_ftp32.zip

After installation of the FTP software, click on the WS_FTP95LE icon to start the application. You should see a main window and a pop-up window for entering and editing connection information. If you don't see the pop-up window, please click "Connect" button on lower-left to activate the pop-up window. In the pop-up window, select "NEW" on the right side and enter the following:

Profile name: Loveocean
Host name / address: 209.52.186.23
User ID: anonymous
Password: (please type any key)

Check the "save pwd" check box. Click "OK" to confirm your selection.

Dial and connect the internet. Start WS_FTP95LE. Under "Profile Name", select "Loveocean", and then click OK. You should be connected to the Loveocean site. The left side of the window is your hard disk location; please select a place you what the downloaded to be stored. The right side is Loveocean site; please select the file you want to download then drag it to the left side of the window (or click <--) to start downloading. If your connection is interrupted in the middle of a downloading, please re-connect, and then the program will continue downloading from where it has been interrupted.
